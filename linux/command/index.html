    linux命令常用用法¶

    awk去重（保持原来顺序）¶

    awk '!a[$0]++' file

    搜索命令保留表头¶

    ps -aux | awk 'NR==1{print} /.*remmina.*/{print}' | grep -v awk

    pigz多线程压缩/解压缩¶

    压缩¶

    tar -cvf - example/ | pigz -p 8 > example.tar.gz

    解压缩¶

    pigz -p 8 -d example.tar.gz

    列出本机所有网络接口及ip¶

    ip addr | egrep "[a-z].*: |inet .*\..*\..*/"| awk '{print $2}'

    nmap测试端口是否开放¶

    TCP¶

    sudo nmap -sS test.example.com -p 12345

    UDP¶

    sudo nmap -sU test.example.com -p 12345

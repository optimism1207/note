    docker¶

    批量清理退出状态的容器¶

    docker rm $(docker ps -qf status=exited)

    以root进入容器¶

    docker run --privileged=true -it alpine:latest /bin/sh

    端口映射与目录挂载¶

    docker run -v 主机目录:容器内目录 -p 主机端口:容器内端口 -it alpine:latest /bin/sh

    进入运行中的容器¶

    exec方式，输exit退出不会对导致容器退出¶

    docker exec -it 容器id /bin/sh

    attach方式，输exit退出会导致容器退出，需使用ctrl+p ctrl+q¶

    docker attach 容器id

    docker镜像体积精简¶

    • 合并多个RUN，清理不需要的残留
    • 添加压缩文件时使用COPY
    • 使用多阶段build直接从之前的阶段拷贝编译生成的文件

    docker普通用户使用¶

    sudo group add docker / sudo useradd docker -s /sbin/nologin
sudo usermod -a -G docker USER
sudo systemctl restart docker
sudo chmod a+rw /var/run/docker.sock
